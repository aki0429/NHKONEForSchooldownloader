# NHK for School Downloader

NHK for Schoolの動画を高画質（MP4）でダウンロードするPythonスクリプトです。公式APIからメタデータを取得し、ffmpegを使用して暗号化されたHLSストリームを自動結合・保存します。

## ✨ 特徴
* **URLを貼るだけ:** 動画のURLを入力するだけで自動でIDを解析し、ダウンロードを開始します。
* **プログレスバー搭載:** ダウンロードの進捗状況（%）と残り時間をコマンドライン上にリアルタイムでスタイリッシュに表示します。
* **403エラー回避:** `Cookie.txt` を読み込むことでブラウザのセッションを引き継ぎ、アクセス制限を確実に突破します。
* **管理者権限不要:** ブラウザの暗号化ファイルを直接触らないため、権限エラーに悩まされずに実行可能です。

## 📦 必須環境
* **Python 3.7 以上**
* **ffmpeg** (コマンドプロンプトから `ffmpeg` コマンドが実行できるよう環境変数のPATHを通しておくこと)
* **必要なPythonライブラリ:** `requests`
